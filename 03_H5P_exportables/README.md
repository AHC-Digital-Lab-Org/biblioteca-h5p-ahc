# 03_H5P_exportables — Paquetes `.h5p` entregables

Contiene los **20 paquetes `.h5p`** que constituyen el entregable principal de la Biblioteca H5P AHC.

**Estado:** ✅ Todos probados y validados en [h5p.org](https://h5p.org) · ⏳ Pendientes de prueba final en Moodle AHC.

Los archivos están nombrados con el esquema:

```
OA-XX_AHC_H5P_[Tipo]_[Tema]_[Nivel]_[Curso]_vX.Y.Z.h5p
```

El prefijo `OA-XX` permite identificar de un vistazo a qué Objeto de Aprendizaje pertenece cada archivo
y los ordena automáticamente por código (OA-01..OA-20) al listar la carpeta.

Para importar en Moodle: *Banco de contenido → Añadir → Subir archivo `.h5p`*.
